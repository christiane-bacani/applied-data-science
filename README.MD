## How to Use This Repository

### Courses
- **AWS AI Practitioner Challenge**: Foundational AWS AI concepts and machine learning fundamentals
- **Kaggle Courses**: Structured learning paths including Python basics and data science introduction

### Mini-Projects
1. **Titanic Competition**: End-to-end machine learning pipeline with data exploration, feature engineering, and model submission
2. **Pandas Loc-Iloc Analysis**: Practical exercises in DataFrame indexing and data manipulation using US YouTube video statistics

## Getting Started
1. Navigate to the course of interest under `courses/` to learn core concepts
2. Review `mini-projects/` for applied examples and complete workflows
3. Each project includes:
   - `datasets/` — Raw data files
   - `notebooks/` — Analysis and implementation notebooks
   - `outputs/` — Generated results and submissions
